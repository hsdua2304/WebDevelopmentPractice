display: flex;
align-items: center;
justify-content: space-around;

align-self: flex-end;
}

align-self: flex-start;
}

Flexbox is a one-dimensional layout for laying out items in rows and columns.

  • Properties : Value
  • Display : flex
  • Flex-wrap : wrap, no-wrap
  • Flex:1
  • Flex:1 200px
  • Flex-direction : row, column, row-reverse, column-reverse
  • Align-self:flex-start,flex-end
